# Джйотиш Астрокартография · Jyotish Astrocartography

Премиальное офлайн-приложение ведической астрокартографии (Джйотиш). По данным рождения строит **сидерическую** натальную карту (аянамша **Лахири**), планетные линии **MC / IC / ASC / DSC** по всему миру, тепловую карту благоприятности, анализ городов и персональные рекомендации для переезда, карьеры, бизнеса, отношений, отдыха и духовной практики.

![brand](https://img.shields.io/badge/zodiac-sidereal%20%C2%B7%20Lahiri-df2227) ![engine](https://img.shields.io/badge/ephemeris-Swiss%20Ephemeris-ef5024)

## Возможности

- **Точная астрономия** — Swiss Ephemeris (`pyswisseph`, режим Moshier) с резервным аналитическим JS-движком, если бэкенд недоступен.
- **Сидерический зодиак, аянамша Лахири** — знаки и накшатры для всех тел.
- **Интерактивная карта мира** (canvas, равнопромежуточная проекция): зум, панорама, линии 9–12 планет в фиксированных цветах, подписи, легенда с тумблерами, тепловая карта.
- **Линии астрокартографии**: MC, IC, ASC, DSC для Солнца, Луны, Марса, Меркурия, Юпитера, Венеры, Сатурна, Раху, Кету (опционально Уран/Нептун/Плутон).
- **Поиск любого города мира** (даже маленькой деревни) через геокодер OpenStreetMap / Nominatim; локальная база ~120 городов — для мгновенных подсказок.
- **Анализ любой точки и города**: расстояние до линий, доминирующие планеты, оценки 0–100 по 14 сферам жизни, текстовая Джйотиш-трактовка.
- **Движок рекомендаций**: лучшие города для ПМЖ / карьеры / бизнеса / отношений / духовности / отдыха / путешествий / цифровых кочевников + «чего избегать».
- **Отчёт** с печатью/PDF, ссылка для шаринга, адаптив (desktop + mobile).

## Запуск

Нужен Python 3 и (для точных эфемерид) пакет `pyswisseph`:

```bash
pip install pyswisseph certifi
python3 server.py
# открыть http://localhost:7899
```

`server.py` отдаёт статику и эндпоинты `/api/chart` (позиции планет) и `/api/geocode` (поиск городов по миру). Без `pyswisseph` приложение работает на встроенном JS-движке; геокодеру нужен интернет, иначе доступна локальная база городов.

## Архитектура

| Файл | Назначение |
|------|-----------|
| `server.py` | Python-сервер: статика + `/api/chart` (Swiss Ephemeris, Лахири) + `/api/geocode` (поиск городов) |
| `js/astro.js` | Резервный JS-движок эфемерид (кеплеровы элементы + лунная теория) |
| `js/acg.js` | Расчёт линий астрокартографии, цвета/глифы планет |
| `js/interp.js` | Аффинности планета↔сфера, оценка локаций, генерация текста |
| `js/map.js` | Canvas-карта: проекция, зум, тепловая карта, хит-тест |
| `js/app.js` | UI: ввод, автодополнение, легенда, панели, рекомендации, отчёт |
| `data/world.geojson` | Контуры мира (упрощённые) |
| `data/cities.js` | База ~120 городов |
| `fonts/jaipur.ttf` | Шрифт заголовков Jaipur |

## Дисклеймер

Приложение предоставляет интерпретации традиционной ведической астрологии (Джйотиш) в образовательных и ознакомительных целях. Результаты не являются гарантиями и не заменяют юридических, финансовых, медицинских или иных профессиональных консультаций.
