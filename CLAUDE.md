# CLAUDE.md — Джйотиш Астрокартография

Офлайн веб-приложение астрокартографии (традиционная индийская астрология, Джйотиш).
Язык интерфейса — русский. Автор: Светлана Кройцер · goroskop1008.ru.

## Рабочий процесс (согласовано)

1. **Сначала обсудить подход** для заметной задачи, потом писать код.
2. **Тестировать локально перед публикацией.** Запуск: `python3 server.py` (порт 7899),
   открыть http://localhost:7899, проверить через preview-инструменты. Расчёт можно
   проверять и через `osascript -l JavaScript` (нет node).
3. **GitHub:** репозиторий `quasisun/astrogeo`. По готовности и проверке — `git commit` + `git push`.
   GitHub Pages: https://quasisun.github.io/astrogeo/ (встраивается в Tilda через iframe).
4. **Issues** (`gh` в `~/bin/gh`): на заметные фичи/баги — `gh issue create` в начале,
   `gh issue close` по готовности. Мелкие правки — без issue.
5. **Кеш:** при изменении js/css **обязательно** поднять версию `?v=N` у всех тегов в `index.html`
   (иначе браузер/Tilda берут старое). Сервер шлёт no-store заголовки.

## Архитектура

| Файл | Назначение |
|------|-----------|
| `server.py` | статика + `/api/chart` (Swiss Ephemeris, аянамша Лахири) + `/api/geocode` (Nominatim) |
| `js/astro.js` | резервный JS-движок эфемерид (если бэкенд недоступен, напр. на GitHub Pages) |
| `js/acg.js` | линии астрокартографии, цвета/глифы планет |
| `js/interp.js` | аффинности планета↔сфера, оценка локаций, генерация русского текста |
| `js/map.js` | canvas-карта: проекция, зум, тепловая карта, точки городов, хит-тест |
| `js/tz.js` | tz-lookup: координаты → зона IANA (смещение считает Intl с учётом DST) |
| `js/app.js` | UI, автодополнение, легенда, панели, рекомендации, отчёт, PDF (через печать) |
| `data/world.geojson` | контуры мира · `data/cities.js` — база ~120 городов |
| `fonts/jaipur.ttf` | шрифт заголовков (только главные заголовки h1/h2; остальное Arial) |

## Соглашения

- Бренд: `#df2227 → #ef5024`, белый фон. Заголовки — Jaipur (h1/h2), текст — Arial.
- Шкалы благоприятности — зелёные. Цвета планет фиксированы (см. `ACG.COLORS`).
- Термин: «традиционная индийская астрология» (не «ведическая»).
- В PDF — никаких ссылок, кроме консультации (`goroskop1008.ru/uslugi#!/tproduct/...`) и копирайта `goroskop1008.ru`.
- Часовой пояс — авто по месту и дате (с учётом перевода часов), с ручным override.
- На GitHub Pages бэкенда нет → расчёт идёт встроенным JS-движком, геокодер обращается к Nominatim напрямую.
